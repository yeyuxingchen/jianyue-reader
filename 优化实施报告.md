## 二、详细变更

### 2.1 安全加固

#### CSP 配置优化
- **文件**: `index.html`
- **状态**: 已优化（无 `unsafe-eval`）
- **说明**: CSP 配置已按照安全最佳实践设置

#### 文件路径验证
- **文件**: `electron/security.ts`
- **功能**: 
  - 默认允许目录: userData, documents, downloads, desktop, temp
  - 用户可添加自定义授权目录
  - 所有文件操作 IPC 均已集成路径验证

#### 敏感数据加密
- **文件**: `electron/security.ts`
- **功能**: 
  - 使用 Electron safeStorage 加密 API Key
  - 支持 Base64 降级方案
  - 已集成到 dbService.ts

#### 错误处理
- **文件**: `electron/errorHandler.ts`
- **功能**:
  - 统一的错误日志记录
  - wrapHandler 包装器用于 IPC 处理器
  - 用户友好的错误对话框

### 2.2 架构重构

#### IPC 模块化拆分
- **目录**: `electron/ipc/`
- **模块列表**:

| 模块 | 文件 | 职责 |
|------|------|------|
| 入口 | `index.ts` | 注册所有处理器 |
| 文件系统 | `fs.ts` | 文件读写操作 |
| 对话框 | `dialog.ts` | 系统对话框 |
| 书籍 | `book.ts` | 书籍缓存管理 |
| 封面 | `cover.ts` | 封面保存/查找 |
| EPUB | `epub.ts` | EPUB 解压/按需加载 |
| 字体 | `font.ts` | 字体管理 |
| AI | `ai.ts` | AI 缓存管理 |
| 图片 | `image.ts` | 图片缓存 |
| 浮动窗口 | `float.ts` | 浮动阅读/笔记窗口 |
| 窗口 | `window.ts` | 窗口控制 |
| Shell | `shell.ts` | 系统操作 |
| 剪贴板 | `clipboard.ts` | 剪贴板操作 |
| 存储 | `store.ts` | electron-store |
| 安全 | `security.ts` | 安全相关 IPC |

#### 窗口工厂
- **文件**: `electron/windowFactory.ts`
- **功能**: 
  - 统一的浮动窗口创建逻辑
  - 事件驱动的数据注入（替代轮询）
  - 降级兼容方案

#### Vue Router
- **文件**: `src/router/index.ts`
- **路由配置**:

| 路径 | 名称 | 组件 | 说明 |
|------|------|------|------|
| `/` | bookshelf | Bookshelf.vue | 书架页 |
| `/reader/:bookId?` | reader | Reader.vue | 阅读页 |
| `/note` | note | NoteEditor.vue | 简记页 |
| `/float/reader` | float-reader | FloatReader.vue | 浮动阅读 |
| `/float/note` | float-note | FloatNote.vue | 浮动笔记 |

#### Composables
- **文件**: `src/composables/`
- **模块**:
  - `useBookAnimation.ts` - 书架动画逻辑
  - `useExternalFileOpen.ts` - 外部文件打开支持

### 2.3 性能优化

#### EPUB 按需加载
- **文件**: `electron/ipc/epub.ts`
- **新增接口**:
  - `epub:listEntries` - 获取文件列表（不解压）
  - `epub:getEntry` - 按需获取单个文件
  - `epub:getEntries` - 批量获取文件
  - `epub:clearCache` - 清理缓存
- **特性**:
  - zip 实例缓存（避免重复打开）
  - LRU 缓存策略（渲染进程）

### 2.4 代码清理

#### 移除的代码
- `_registerLegacyIpcHandlers_DEPRECATED()` 函数（~800 行）
- 未使用的 `epubjs` 依赖

#### 新增的文件
| 文件 | 说明 |
|------|------|
| `electron/config.ts` | 共享配置和工具函数 |
| `electron/windowFactory.ts` | 窗口工厂 |
| `src/router/index.ts` | 路由配置 |
| `src/composables/useBookAnimation.ts` | 书架动画 composable |
| `src/composables/useExternalFileOpen.ts` | 外部文件打开 composable |
| `src/composables/index.ts` | composables 导出 |
