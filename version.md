# 版本更新日志

## 简阅 v1.1.0

基于 v1.0.0 的重大功能更新版本，新增 AI 对话、TTS 语音朗读、EPUB 导出等核心能力。

---

### 新增功能

#### AI 对话助手
- 接入 OpenAI 兼容 API，支持自定义接口地址与 API Key
- 流式对话输出，实时显示回复内容
- 支持多模型配置与切换（逗号分隔配置多个模型）
- 回复字数限制（50/100/200/500 字可选）
- 自动携带阅读背景上下文（书名、作者、目录、当前章节）
- 对话历史按书籍独立保存，支持随时回看
- 可从选中文本直接发起 AI 搜索解释
- 对话窗口支持拖拽移动与自由缩放

#### TTS 语音朗读
- 支持多种音色：冰糖、苏打、茉莉、白桦、Mia、Chloe、Milo、Dean 等
- 28 种朗读风格，覆盖文学、小说、人文社科三大分类
  - 文学：经典文学、当代文学、散文随笔、诗歌、戏剧、纪实文学、文学评论
  - 小说：正剧、轻松搞笑、暗黑压抑、治愈温暖、热血燃、文艺唯美、爽文、虐文、种田日常、升级流、群像、单元剧
  - 人文社科：历史、哲学、宗教、社会学、人类学、政治学、法学、经济学、管理学、心理学
- 支持自定义朗读风格提示词
- 逐句高亮同步，朗读进度实时可视
- 智能预取队列（1-5 句可配），播放流畅不卡顿
- 自动分句与音频标签注入（激动/疑惑/停顿）
- 章节结束自动暂停，可继续朗读下一章
- 支持跳转指定句子继续播放

#### 笔记编辑器增强
- **代码块语法高亮**：支持 100+ 编程语言，基于 Prism.js + Refractor
- **代码块语言切换器**：光标所在代码块右下角快速切换语言
- **文件树管理**：侧边栏文件目录面板，支持章节/目录新建、重命名、删除
- **EPUB 目录自动收敛**：打开 EPUB 解压目录时自动定位到内容子目录
- **章节图片插入**：支持粘贴/选择图片插入章节，自动管理 .image 图片资源
- **EPUB 导出**：笔记目录一键导出为 EPUB 3.0 电子书，支持封面设置
- **自动保存**：已知路径文件修改后自动保存到磁盘（防抖 2s），持续编辑不打断
- **源码模式大纲跳转**：源码编辑模式下支持标题大纲快速定位
- **文字缩放**：Ctrl + 滚轮调整编辑器文字大小（50%~200%）
- **悬浮笔记代码块样式**：浮窗 Markdown 代码块添加配色与自定义滚动条

#### 系统与体验
- **系统文件关联**：安装后自动关联 `.epub` `.txt` `.mobi` `.azw3` `.cbz` `.cbr` `.md` 文件，双击即可用简阅打开
- **系统托盘**：关闭窗口自动最小化到托盘，托盘右键菜单（显示窗口/关于/退出）
- **安全加固**：
  - API Key 等敏感数据使用系统级加密存储（Electron safeStorage）
  - 文件访问授权目录管理，限制文件操作范围
- **多架构安装包**：
  - Windows：x64 / ia32 / arm64
  - Linux：x64 / arm64（AppImage / deb）
  - macOS：x64 / arm64（dmg）
- **NSIS 自定义安装脚本**：安装详情框输出已安装文件列表
- **侧边栏面板动效**：面板展开收起添加平滑过渡动画
- **通用 Tooltip 组件**：替换侧边栏原生 title，统一视觉风格
- **模式切换保护**：简阅/简记模式切换时自动检测未保存更改并弹窗确认
- **外部打开优化**：双击文件打开去重，书架数据预加载

## 简阅 v1.0.0

本地电子书阅读器与笔记工具，首次发布。

### 阅读功能
- 支持 EPUB / TXT / MOBI / AZW3 / CBZ / CBR 格式
- 12 套阅读主题（日间 / 夜间 / 护眼）
- 滚动 / 单页双模式
- 多色高亮划线与备注系统
- 书签管理与全文搜索
- 进度记忆、字体排版调节

### 笔记功能
- 基于 Milkdown 的所见即所得 Markdown 编辑器
- 从阅读器提取文字生成浮动笔记窗口
